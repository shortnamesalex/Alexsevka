Alexsevka is a custom build of Iosevka with a strong focus on readability and a contemporary yet futuristic aesthetic that would look at home in the den of a netrunner, ready to make their big break. It takes inspiration from Jetbrains Mono and Fraktion Mono, using Iosevka's wide array of customization options to incorporate the former's legible glyphs and the latter's techy and punk-like attitude.

![alexsevka_banner_1](https://github.com/user-attachments/assets/bf1d1731-0f04-4b15-9985-6fbb0a81cb2a)

The major differences from Iosevka include the new default width of ```600``` and most glyphs using a more "neutral" design. It has the same extensive character set as Iosevka—covering over 200 languages + a large collection of ligatures. Several Cyrillic glyphs have been adjusted to match the Latin counterpart and the switch to obliques. Alexsevka also comes with 4 widths and 7 weights, aiming to be a bit more than just a programming font.
